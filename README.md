
# 📊 Dynamic Heat Map Using Excel

## Overview

This project demonstrates the creation of a dynamic heat map in Microsoft Excel to analyze accident patterns in the city of Cambridge.

The dashboard helps visualize accident frequency across different time intervals, seasons, and accident types, enabling users to identify high-risk and low-risk periods for travel.

🎯 Objective
Analyze accident data using Excel
Identify patterns based on:
Time of day
Day of the week
Accident type
Seasonal variations
Build an interactive and dynamic heat map dashboard
Provide geographic visualization using Power Map
📌 Key Features
🔄 Fully dynamic heat map updated via dropdown filters
🎛️ Interactive filtering by:
Accident type
Season
📊 Pivot table-driven data model
🎨 Conditional formatting-based heat visualization
🗺️ Geo heat map visualization using Power Map (3D Maps)
📈 Real-time updates of:
Total accident count
X-axis (time/day)
Y-axis (day/time patterns)
🧰 Tools & Technologies Used

Slicers (if applicable)

Data Modeling

🛠️ How to Use
Open the Excel file
Go to the Heat Map worksheet
Locate the dropdown filters at the top
Select:
Accident Type
Season Type
The heat map will automatically update based on your selection
Observe:
Changes in heat map intensity
Updated total accident count
Dynamic X and Y axes
Geo heat map updates at the bottom
